O que é python
Python é uma linguagem de programação de alto nível, interpretada e de propósito geral. Reconhecida por sua
sintaxe simples e legibilidade, é amplamente utilizada em desenvolvimento web, científico, análise de dados,
automação e muito mais. Sua comunidade ativa e vasta biblioteca de módulos contribuem para sua popularidade e versatilidade.


Tutorial de instalação do python

Primeiro passo:
acesse o site oficial do python (Python.org)
lá aparecerá um botão de download já para baixar o instalador da ultima versão
clique em download para baixar o instalador da ultima versão

Segundo passo:
localize o instalador e clique duas vezes sobre ele
assim que o instalador abrir clique em adicionar o python ao PATH
e install now

Terceiro passo:
após a instalação verifique a instalação
abra o prompt de comando do windowns e de o comando(python --version)
isso irá exibir a versão que acabou de instalar do python
